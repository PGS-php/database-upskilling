DATABASE UPSKILLING
===================
Simple MySQL 5.7 provisioning based on Docker with test database based on repository [datacharmer/test_db](https://github.com/datacharmer/test_db) 

### Database schema 
![Schema](schema.png)

### Getting started
 1. Build image:
 ```bash
docker-compose build
```
 2. Run container:
 ```bash
docker-compose up
```
